## 使用前说明

请在 [LSPosed](https://github.com/LSPosed/LSPosed/releases) 中启用，给予软件**ROOT**权限，即可在应用中启用对应的功能，并强制重启该应用生效

在`第三方 Flyme ROM`与`国际版Flyme ROM`支持行为可能严重受损

我们不断同步基于最新的官方**MEIZU 21 Flyme12**内测其他机型版本`可能存在BUG与不兼容行为`
